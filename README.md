# Social Media App

A full-stack social media application built with the MERN stack (MongoDB, Express.js, React, Node.js).

## Features

- 🔐 User Authentication (Login/Register)
- 📝 Create Posts
- ❤️ Like Posts
-  Comment on Posts
- 👤 User Profiles
- 📱 Responsive Design

## Tech Stack

**Backend:**
- Node.js
- Express.js
- MongoDB
- JWT for Authentication
- Bcrypt for Password Hashing

**Frontend:**
- React
- React Router
- CSS3
